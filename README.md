// TWEAK 1: CPU Priority para High

## 📥 **Como Compilar**

### 🔧 **Pré-requisitos**

- **.NET 8.0 SDK** ou superior
- **Visual Studio 2022** ou Visual Studio Code
- **Windows 10** (1903+) ou **Windows 11**
- **Git** para clonar o repositório

### 🚀 **Método 1: Visual Studio**

1. **Clone** o repositório:
   ```bash
   git clone https://github.com/luigiarrud4/KitLugia.git
   cd KitLugia
   ```

2. **Abra** o arquivo `KitLugia.sln` no Visual Studio

3. **Selecione** a configuração `Release`

4. **Build** a solução (Ctrl+Shift+B)

5. **Execute** o projeto `KitLugia.GUI`

### ⚡ **Método 2: .NET CLI**

1. **Clone** o repositório:
   ```bash
   git clone https://github.com/luigiarrud4/KitLugia.git
   cd KitLugia
   ```

2. **Build** o projeto:
   ```bash
   dotnet build --configuration Release
   ```

3. **Execute** a aplicação:
   ```bash
   cd KitLugia.GUI/bin/Release/net8.0-windows
   KitLugia.GUI.exe
   ```

### 📦 **Build para Distribuição**

```bash
# Build self-contained (sem necessidade de .NET runtime)
dotnet publish KitLugia.GUI -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true

# Output em: KitLugia.GUI/bin/Release/net8.0-windows/win-x64/publish/
```
